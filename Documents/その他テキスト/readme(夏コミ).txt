// はじめに
このゲームは、東方Project(上海アリス幻樂団)の二次創作です。

// ゲーム及び今バージョンについて
このゲームは、アリスの操る人形たちを合成・強化しながら
戦略性豊かなバトルでステージを攻略するシミュレーションRPGです…が、
現在まだ制作の序盤であり、今回、夏コミ用として
戦闘システムだけを最小限ながらある程度遊べるように用意しました。
内容は今後のバージョンで大幅に変わる可能性があります。

// 操作
十字キー：コマンドを選択します。
Zキー：コマンドを決定します。
Xキー：コマンドをキャンセルします。
Aキー：現在のコマンドを説明するウィンドウを開きます。

// 戦闘システム概要
1. 形式
多対多のターン制のバトルです。
ターンごとにコマンドを選択し、素早さの高いキャラクターから順番に行動します。
相手を全滅させたら勝利、味方が全滅したら敗北です。
本来はステージ攻略型ですが、今回のバージョンでは戦闘一回ごとにメニューに戻ります。
また、本来はこちら側は「前衛」三体とその他の「後衛」に分かれており、前衛のみが戦います。
後述する属性に合わせて人形の交代などが可能になっていますが、
今回は残念ながら実装ができていません。

2. 属性システム
人形や敵の多くは属性を持ちます。
属性は、太陽、月、星の三種類で、属性を持たないキャラも居ます。
太陽は月に、月は星に、星は太陽に強く、
有利な属性の敵に対してはダメージが2倍、不利な属性の敵に対してはダメージが半分になります。
属性の相性がかなり重要となるシステムなので、
不利な敵の攻撃を受けないよう、また相手に有効なダメージを与えられるよう、
特技などをうまく使って立ち回りましょう。また、今回実装を見送った前衛・後衛の交換も
不利な属性を回避するための主要なシステムです。

3. パラメータ
人形及び敵キャラの、戦闘に関わるパラメータは以下の通りです。
HP：体力です。ダメージを受けると減少し、0になると戦闘不能になります。
攻撃：攻撃で相手に与えるダメージです。特技の威力にも影響します。
防御：攻撃に対する抵抗力です。人形にはこのパラメータはありません。
敏捷：このパラメータが大きいキャラクターから行動します。一部特技の威力にも影響します。
魔力：一部の特技の威力に関するパラメータです。
技巧：一部の特技の威力に関するパラメータです。
通常攻撃によるダメージは以下のように計算されます。
敵に与えるダメージ：人形の攻撃−敵の防御
人形が受けるダメージ：敵の攻撃
人形のパラメータは、コマンド選択時にAキーを押すと確認できます。
敵のパラメータは、「今バージョンでは」ターゲット選択時にAキーを押すと確認できます。
今後、「倒した敵のみ確認可能」などに変更する予定です。
